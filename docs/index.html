This tool is an unofficial IPv6 helper. It derives an IPv6 link-local address from a MAC address using the EUI-64 method.

Always verify results on the actual device and with official documentation before making configuration changes or relying on connectivity assumptions. The author assumes no liability for misconfiguration, downtime, or any resulting damage.

MAC → IPv6 Link-Local (EUI-64) Calculator

Enter a MAC address to derive the corresponding fe80::/64 link-local IPv6 address.

Accepted formats: 00-1A-2B-3C-4D-5E, 00:1A:2B:3C:4D:5E, 001A.2B3C.4D5E, 001A2B3C4D5E.
